## 🎯 Tips for Better Results

### ✅ DO:
- Be specific about what you want (e.g., "winter clothing", "women's coats")
- Include style keywords: "professional", "e-commerce", "studio lighting"
- Mention background: "white background", "clean background"
- Add quality terms: "high quality", "detailed", "sharp focus"

### ❌ DON'T:
- Use vague terms like "nice image" or "good picture"
- Forget to specify the style you want
- Use too many conflicting styles in one prompt

## 📝 Blog Writing Tips

### ✅ DO:
- **Be specific:** Mention the topic, target audience, and desired tone
- **Include structure:** Ask for sections, headings, or outline if needed
- **Specify length:** Mention if you want a short post (500-800 words) or long-form (1500+ words)
- **Add context:** Include your target audience (beginners, professionals, general readers)
- **Request examples:** Ask for real-world examples, case studies, or actionable tips

### ❌ DON'T:
- Use vague requests like "write a blog about business"
- Forget to specify your target audience
- Skip mentioning the desired tone (professional, casual, informative, etc.)

## 🔥 Best Blog Prompts Format

**For Comprehensive Articles:**
```
Write a detailed blog article about [TOPIC] for [TARGET AUDIENCE]. 
Include [SPECIFIC SECTIONS], provide [EXAMPLES/CASE STUDIES], 
and make it [TONE: engaging/informative/professional]. 
The article should be approximately [WORD COUNT] words.
```

**For How-To Guides:**
```
Create a step-by-step guide on [TOPIC] for [AUDIENCE]. 
Include practical tips, common mistakes to avoid, and 
actionable advice. Make it easy to follow and implement.
```

**For Opinion Pieces:**
```
Write a thought-provoking blog post about [TOPIC] from 
[PERSPECTIVE]. Include your analysis, supporting evidence, 
and recommendations. Target audience: [AUDIENCE].
```

## 🎯 Blog Article Structure Suggestions

When requesting blog articles, you can also specify structure:

1. **Introduction** - Hook the reader
2. **Problem/Context** - Set the stage
3. **Main Content** - Detailed information
4. **Examples/Case Studies** - Real-world applications
5. **Actionable Tips** - Practical advice
6. **Conclusion** - Summary and call-to-action

**Example:**
```
Write a blog article about [TOPIC] with the following structure:
1. Engaging introduction with a hook
2. Problem statement or context
3. Main content with detailed explanations
4. Real-world examples or case studies
5. Actionable tips and takeaways
6. Strong conclusion with call-to-action
Target: [AUDIENCE], Tone: [TONE], Length: [WORDS]
```
